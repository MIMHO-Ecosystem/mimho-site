On-Chain Rules

Competition parameters are enforced by the contract: duration, minimum thresholds, eligibility, and scoring outputs. This prevents off-chain favoritism.

Anti-Abuse Mindset

The module is designed to reduce obvious farming patterns and suspicious loops without blocking honest participants. Transparency is the default.

HUD + Events Hub

Every key milestone can be broadcast to the Events Hub for the MIMHO HUD: start announcements, cycle creation, and public results.

Documentation Overview

This page explains what Trading Activity is, how competitions are started, and how users can verify results. The goal is simple: competitions with rules you can inspect, not rules you have to trust.
